# Pants Planner

This is a minimal prototype for the **Pants Edition** to-do list and daily planner. It is a simple mobile-friendly web app that stores tasks in your browser's local storage.

## Features

- Task List and Planner views
- Add tasks with title, estimated time, priority, and optional hour assignment
- Hour-by-hour planner displaying tasks assigned to each hour
- Sorting by priority
- Data persisted locally in the browser

## Running

Open `src/index.html` in a modern browser. No build steps are required.
